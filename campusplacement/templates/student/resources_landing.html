1. Numbers & LCM/HCF

Key Formulas:
• Product of two numbers = Product of their HCF and LCM.
• HCF of Fractions = (HCF of Numerators) / (LCM of Denominators)
• LCM of Fractions = (LCM of Numerators) / (HCF of Denominators)

2. Time, Speed & Distance

• Speed = Distance / Time
• Average Speed = (2xy) / (x + y) (for equal distances at speeds x and y)
• Relative Speed (Same Direction) = u - v
• Relative Speed (Opposite Direction) = u + v

3. Profit, Loss & Discount

• Profit % = (Profit / CP) * 100
• SP = ((100 + Gain%) / 100) * CP
• Discount = Marked Price - Selling Price

4. Permutations & Combinations

• Permutation (Arrangement): nPr = n! / (n-r)!
• Combination (Selection): nCr = n! / (r! * (n-r)!)
• Circular Permutation: (n-1)! for distinct objects

5. Probability

• P(E) = Favorable outcomes / Total possible outcomes
• Mutually Exclusive: P(A or B) = P(A) + P(B)
• Independent Events: P(A and B) = P(A) * P(B)

6. Geometry & Mensuration

• Circle: Area = πr², Circumference = 2πr
• Cylinder: Volume = πr²h, Surface Area = 2πrh + 2πr²
• Triangle: Area = ½ * base * height or √(s(s-a)(s-b)(s-c))

Logical Reasoning

1. Blood Relations

Focus on family trees. Draw relationships generation by generation.

Common Symbols: Square (Male), Circle (Female), Single Line (Siblings), Double Line (Spouse).

2. Syllogisms

Use Venn Diagrams to validate conclusions. Remember: "Some A are B" does not always mean "Some B are NOT A".

Keywords: All, Some, None, Some Not. Always draw the minimal overlapping case to avoid false positives.

3. Coding-Decoding

Alphabetic Positioning: A=1, E=5, J=10, O=15, T=20, Y=25 (EJOTY mnemonic). Always check for reverse positioning (Z=1, Y=2...).

4. Seating Arrangements

Linear, Circular, and Matrix arrangements. Always fix the most constrained position first.

Tip: For circular facing center, Left = Clockwise, Right = Anti-clockwise. Facing outside is the exact opposite.

5. Clocks & Calendars

• Angle between hands: |(30 * H) - (5.5 * M)|
• Leap Year: Divisible by 4 (and 400 if century year).
• Odd Days: Remainder when total days are divided by 7.

6. Data Sufficiency

Don't solve the problem! Just determine if the provided statements contain enough information to solve it.

Checklist: 1. Check A alone, 2. Check B alone, 3. Check A and B together.

7. Direction Sense

• Pythagoras Theorem: c² = a² + b² (for finding shortest distance)
• Sun sets in the West, rises in the East (shadow problems).

8. Number & Letter Series

• Common patterns: Prime numbers, alternating series, squares/cubes ± n.
• Fibonacci: 0, 1, 1, 2, 3, 5, 8, 13...

9. Venn Diagrams

• n(A U B) = n(A) + n(B) - n(A ∩ B)
• n(A U B U C) = n(A) + n(B) + n(C) - n(A ∩ B) - n(B ∩ C) - n(C ∩ A) + n(A ∩ B ∩ C)

Data Structures & Algorithms

1. Array & Strings

Two Pointers, Sliding Window, Prefix Sum algorithms.

// Reverse an array in O(n/2) void reverse(int arr[], int n) { for(int i=0; i < n/2; i++) { swap(arr[i], arr[n-1-i]); } }

2. Complexity Analysis

• Binary Search: O(log n)
• Quick Sort (Avg): O(n log n)
• Merge Sort (All): O(n log n)
• Hash Table: O(1) Avg

3. Linked Lists

Fast & Slow Pointers, Reversal, Merge.

// Detect Loop in Linked List (Floyd's Cycle) bool hasCycle(Node* head) { Node *slow = head, *fast = head; while(fast && fast->next) { slow = slow->next; fast = fast->next->next; if(slow == fast) return true; } return false; }

4. Trees & Graphs

Traversals (Inorder, Preorder, Postorder, BFS, DFS), Shortest Path, MST.

• BFS uses Queue. DFS uses Stack (or Recursion).
• Dijkstra's Algorithm: O((V+E) log V)
• BST Property: Left Child < Parent < Right Child

5. Dynamic Programming

Overlapping Subproblems and Optimal Substructure. Memoization vs Tabulation.

Classic Problems: Fibonacci, 0/1 Knapsack, Longest Common Subsequence (LCS), Coin Change.

6. Sorting & Searching

Know when to use which algorithm based on stability, space constraints, and data characteristics.

• Binary Search requires sorted array.
• Merge Sort is stable. Quick Sort is in-place.
• Counting Sort is O(N) but requires limited range.

7. Stacks & Queues

• Stack: LIFO (Last In First Out). Uses: Valid Parentheses, Infix to Postfix.
• Queue: FIFO (First In First Out). Uses: BFS, task scheduling.

8. Hashing & Heaps

• Hash Table: O(1) average. Collision resolution: Chaining, Open Addressing.
• Min Heap / Max Heap: Parent is always smaller/larger than children. Used for Priority Queues.

9. Bit Manipulation

• n & (n-1): Clears the lowest set bit.
• a ^ a = 0 (XOR). a ^ 0 = a.
• Left shift: a << 1 (multiply by 2), Right shift: a >> 1 (divide by 2).

CS Core Essentials

1. Database Management (DBMS)

ACID Properties: Atomicity, Consistency, Isolation, Durability.
Normal Forms: 1NF (Atomic), 2NF (No partial dependency), 3NF (No transitive dependency), BCNF.
Keys: Primary, Foreign, Candidate, Super, Alternate.
Indexing: B-Trees, B+ Trees, Hashing to optimize retrieval speed.

2. Operating Systems (OS)

Process Management, Memory Management, and Concurrency.

Deadlock Conditions: Mutual Exclusion, Hold and Wait, No Preemption, Circular Wait.
Scheduling Algorithms: FCFS, SJF, Round Robin, Priority Scheduling.
Memory: Paging, Segmentation, Virtual Memory, Thrashing.

3. Computer Networks (CN)

OSI Layers: Physical, Data Link, Network, Transport, Session, Presentation, Application.
TCP vs UDP: TCP is reliable, connection-oriented. UDP is fast, connectionless.
Key Protocols: HTTP/HTTPS, DNS, IP, ARP, ICMP, DHCP.
Subnetting: IPv4 addressing, CIDR notation, subnet masks.

4. Software Engineering

SDLC Models: Waterfall, Agile, Scrum, DevOps.
Testing: Unit, Integration, System, Acceptance, Regression.
SOLID Principles: Single Responsibility, Open-Closed, Liskov Substitution, Interface Segregation, Dependency Inversion.

5. System Design Basics

Scaling: Vertical (scale-up) vs Horizontal (scale-out).
Load Balancing: Distributing traffic across multiple servers.
Caching: Redis, Memcached. Reduces DB load.
Microservices: Decoupled, independent deployable services vs Monolith.

6. Object Oriented Programming (OOP)

Pillars: Encapsulation, Abstraction, Inheritance, Polymorphism.
Access Modifiers: Public, Private, Protected.
Overloading vs Overriding: Compile-time vs Run-time polymorphism.

7. Cloud Computing & DevOps

IaaS, PaaS, SaaS: Infrastructure, Platform, Software as a service.
CI/CD: Continuous Integration (Auto tests) / Continuous Deployment (Auto releases).
Containers: Docker (lightweight virtualization), Kubernetes (orchestration).

8. Web Security & Cryptography

Vulnerabilities: SQL Injection (un-sanitized inputs), XSS (cross-site scripting), CSRF.
Encryption: Symmetric (same key) vs Asymmetric (Public/Private key).
Hashing: One-way transformation (SHA-256, bcrypt).

9. REST APIs & Architecture

HTTP Methods: GET (Read), POST (Create), PUT/PATCH (Update), DELETE (Remove).
Status Codes: 200 (OK), 400 (Bad Request), 401 (Unauthorized), 404 (Not Found), 500 (Server Error).
Statelessness: Server stores no client context between requests.

HR Interview Preps

Q: "Tell me about yourself."

Target: 60-90 seconds. Structure: Present -> Past -> Future. Mention your current role/education, highlight 2-3 significant past achievements/projects, and end with why you're excited about this specific opportunity.

Q: "Why should we hire you?"

Target: Align your specific skills with the job description. Give a concrete example of how your unique combination of skills solves a problem the company is currently facing.

Q: "What is your greatest weakness?"

Target: Mention a real but non-critical weakness, and most importantly, show the actionable steps you are taking to improve it. Never say "I'm a perfectionist".

Q: "Where do you see yourself in 5 years?"

Target: Show ambition but focus on learning and contributing to the company's growth. Express a desire for increased responsibility and expertise in your field.

Behavioral Questions: The STAR Method

Use this to answer "Tell me about a time when..." questions.

  • Situation: Set the scene and give necessary context.
  • Task: Describe your specific responsibility or the challenge.
  • Action: Explain exactly what YOU did (use "I", not "we").
  • Result: Share the positive outcome, using metrics if possible.

Q: "What are your salary expectations?"

Target: Deflect if early in the process ("I'm currently focused on finding the right fit"). If pressed, provide a well-researched range based on the market and your skills, stating you're open to negotiation based on total compensation.

Q: "Do you have any questions for us?"

Target: ALWAYS say yes. Ask about the team culture, the biggest challenges the team is facing right now, or the typical career path for this role. E.g., "What does a successful first 30 days look like in this role?"

Q: "Tell me about a time you failed."

Target: Own the failure without blaming others. Focus 20% on the failure itself and 80% on what you learned and the systemic changes you made to prevent it from happening again.

Q: "How do you handle conflict in a team?"

Target: Show empathy and communication skills. Mention active listening, addressing the issue privately, focusing on the shared goal, and finding a professional compromise.

1. PHP Basics – Variables, Datatypes & Constants

Variable Rules: Always start with $. Case-sensitive. No type declaration needed.
Datatypes: String, Integer, Float, Boolean, Array, Object, NULL.
Constants: define("PI", 3.14); or const MAX = 100; – global scope, no $.
Magic Constants: __LINE__, __FILE__, __DIR__, __FUNCTION__, __CLASS__.

2. Operators & Control Flow

Operators: Arithmetic (+,-,*,/,%), Assignment (=,+=,-=), Comparison (==,===,!=,<>), Logical (&&,||,!), Ternary (?:), Null coalescing (??).
Control: if/elseif/else, switch/case, match (PHP 8+).
Loops: for, while, do-while, foreach (best for arrays).
Jump: break, continue, return.

3. Functions & Regular Expressions

Functions: function name($arg = default) { return ...; }
Variable-length args: ...$args (splat operator).
Regex: preg_match(), preg_replace(), preg_split().
Common patterns: /^[a-z]+$/i, /\d{10}/, /\b\w+@\w+\.\w+\b/.

4. Arrays & SuperGlobals

Indexed: $a = [1,2,3];   Associative: $a = ['key'=>'val'];
Key functions: array_push, array_pop, array_merge, array_keys, array_values, in_array, array_map, array_filter.
SuperGlobals: $_GET, $_POST, $_SESSION, $_COOKIE, $_SERVER, $_FILES, $GLOBALS.

5. Object-Oriented PHP (OOP)

Pillars: Encapsulation (access modifiers: public/protected/private), Inheritance (extends), Polymorphism, Abstraction (abstract class / interface).
Constructor: function __construct(...) {}
Static: self::$var, ClassName::method().
Traits: horizontal code reuse across unrelated classes.

6. PHP with MySQL – CRUD

Connect (MySQLi): $conn = new mysqli($host,$user,$pass,$db);
Query: $res = $conn->query("SELECT * FROM table");
Fetch: $row = $res->fetch_assoc();
Prepared Statements: Use prepare/bind_param/execute to prevent SQL injection.
PDO: Database-agnostic; use named placeholders :name.

7. Sessions & Cookies

Session: session_start(); then $_SESSION['key'] = val; Destroy: session_destroy();
Cookie: setcookie(name, value, expiry, path); Read: $_COOKIE['name'].
Difference: Sessions stored server-side (secure); cookies stored client-side (persist across browser close).

8. HTTP Methods – GET vs POST

GET: Data in URL query string; bookmarkable; max ~2 KB; idempotent – use for reads.
POST: Data in request body; not cached; no size limit; use for writes/sensitive data.
Form embedding: <form method="POST" action="handler.php">.
Embedding PHP: <?php echo $var; ?> or short echo <?= $var ?>.

1. Introduction to AI – Agents & Environments

Agent: Perceives environment via sensors, acts via actuators.
PEAS: Performance measure, Environment, Actuators, Sensors.
Environment types: Fully/Partially observable, Deterministic/Stochastic, Episodic/Sequential, Static/Dynamic, Discrete/Continuous.
Agent types: Simple Reflex → Model-based → Goal-based → Utility-based → Learning Agent.

2. Problem Solving – Search Algorithms

Uninformed: BFS (Complete, Optimal for unit cost), DFS (Not optimal), DLS, IDDFS, UCS.
Informed (Heuristic): Greedy Best-First (f = h), A* (f = g + h), IDA*.
Admissibility: h(n) ≤ true cost → A* is optimal.
Local Search: Hill Climbing, Simulated Annealing, Genetic Algorithms (for large-space optimisation).

3. Knowledge Representation & Reasoning

Propositional Logic: AND (∧), OR (∨), NOT (¬), Implication (→), Biconditional (↔).
First-Order Logic (FOL): Predicates, Quantifiers (∀, ∃), Functions.
Inference: Modus Ponens, Resolution, Unification.
Semantic Nets & Frames: Graph-based KR structures.

4. Machine Learning – Core Concepts

Types: Supervised (labelled data), Unsupervised (clustering), Reinforcement (reward-based).
Key Algorithms: Linear/Logistic Regression, Decision Trees, SVM, k-NN, Naive Bayes, k-Means, Random Forest.
Bias-Variance: High bias = underfitting; High variance = overfitting.
Evaluation: Accuracy, Precision, Recall, F1-Score, ROC-AUC.

5. Data Science Lifecycle

Steps: Problem Definition → Data Collection → EDA → Preprocessing → Modelling → Evaluation → Deployment.
EDA Tools: Pandas (DataFrames), Matplotlib/Seaborn (visualisation), NumPy (arrays).
Preprocessing: Handling nulls, encoding categoricals, feature scaling (StandardScaler, MinMaxScaler), feature engineering.

6. Data Mining & Warehousing

Data Mining: KDD process – Selection → Preprocessing → Transformation → Mining → Interpretation.
Techniques: Classification, Clustering, Association Rules (Apriori – support, confidence, lift).
Data Warehouse: OLAP (Online Analytical Processing) vs OLTP. Star schema, Snowflake schema, Fact & Dimension tables.
ETL: Extract, Transform, Load.

7. Deep Learning & Neural Networks

ANN: Input → Hidden layers → Output. Activation: ReLU, Sigmoid, Softmax.
Backpropagation: Chain rule to compute gradients; optimised by SGD / Adam.
CNN: Convolutional layers for image data (edge/pattern detection).
RNN/LSTM: Sequential data – NLP, time-series.

8. Natural Language Processing (NLP)

Pipeline: Tokenisation → Stopword removal → Stemming/Lemmatisation → Vectorisation → Model.
Vectorisation: Bag of Words, TF-IDF, Word2Vec, BERT embeddings.
Tasks: Sentiment Analysis, Named Entity Recognition (NER), Machine Translation, Text Summarisation.

1. Introduction to Digital Marketing

Definition: Promotion of products/services through digital channels – search engines, social media, email, websites.
Key Channels: SEO, SEM/PPC, Social Media Marketing (SMM), Content Marketing, Email Marketing, Affiliate Marketing, Influencer Marketing.
Advantages over traditional: Measurable ROI, targeted audience, real-time analytics, cost-effective, global reach.

2. Search Engine Optimisation (SEO)

On-Page SEO: Title tags, meta descriptions, heading hierarchy (H1-H6), keyword density, alt text, URL structure.
Off-Page SEO: Backlinks (quality over quantity), Domain Authority (DA), social signals.
Technical SEO: Site speed, mobile-friendliness, sitemap.xml, robots.txt, HTTPS, Core Web Vitals.
Tools: Google Search Console, Ahrefs, SEMrush, Moz, Screaming Frog.

3. Search Engine Marketing (SEM) & PPC

PPC (Pay-Per-Click): Pay only when a user clicks the ad (Google Ads, Bing Ads).
Ad Rank = Bid × Quality Score.
Quality Score factors: Expected CTR, Ad relevance, Landing page experience (1-10 scale).
Metrics: CTR, CPC, CPM, CPA, ROAS, Impression Share, Conversion Rate.

4. Social Media Marketing (SMM)

Platforms: Facebook/Instagram (Meta Ads), YouTube, LinkedIn (B2B), Twitter/X, Pinterest, Snapchat.
Organic vs Paid: Organic = free posts; Paid = boosted/sponsored content with audience targeting.
Content Types: Reels, Stories, Carousels, Lives, Polls, UGC (User Generated Content).
Engagement Rate = (Likes + Comments + Shares) / Reach × 100.

5. Content Marketing

Content Funnel: Awareness (blogs, infographics) → Consideration (case studies, webinars) → Decision (demos, testimonials).
Content Calendar: Plan topics, formats, publishing dates, and distribution channels.
SEO Content: Long-tail keywords, pillar pages, topic clusters, internal linking.
KPIs: Page views, Time on page, Bounce rate, Leads generated.

6. Email Marketing

List Building: Opt-in forms, lead magnets (ebooks, webinars), GDPR-compliant consent.
Segmentation: By demographics, behaviour (clicks, purchases), lifecycle stage.
Metrics: Open Rate, Click-Through Rate (CTR), Unsubscribe Rate, Deliverability, Conversion Rate.
Tools: Mailchimp, SendGrid, HubSpot, ActiveCampaign.

7. Web Analytics & Data-Driven Marketing

Google Analytics 4 (GA4): Event-based model. Key metrics – Sessions, Users, Engagement Rate, Conversions.
Key Concepts: Bounce Rate, Average Session Duration, Pages/Session, Goal Completion.
UTM Parameters: Track campaign source, medium, and name in URLs.
A/B Testing: Test one variable at a time; sufficient sample size needed for statistical significance.

8. E-Commerce & Affiliate Marketing

E-Commerce Funnels: Visit → Product View → Add to Cart → Checkout → Purchase → Retention.
Cart Abandonment: Avg ~70%; recover with retargeting ads, abandoned-cart emails.
Affiliate Marketing: Publisher earns commission per sale/lead through a tracked affiliate link.
Influencer Tiers: Nano (<10K), Micro (10-100K), Macro (100K-1M), Mega (>1M) followers.
